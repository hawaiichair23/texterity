Texterity Script API reference — every function, param, and default.

Text Objects

createTextObject (config) → id

Creates a new text object on the overlay. If a text object with the same name already exists, it reuses it instead of creating a duplicate — safe to call on every script run.

Param Default Description
name — Unique name. Used to reuse the object on re-run.
text 'New Text' The displayed string.
x center Horizontal position in pixels.
y center Vertical position in pixels.
fontSize 48 Font size in pixels.
fontFamily 'Instrument Serif' Font name.
color 0xFFFFFF Text fill color as hex number.
outlineEnabled true Whether to show a text outline.
outlineColor 0x000000 Outline color.
outlineWidth 4 Outline thickness. 
const myText = createTextObject({
  name: 'My Text',
  text: 'Hello World',
  x: 640,
  y: 360,
  fontSize: 60,
  color: 0xFFFFFF
});
updateTextObject (id, updates)

Updates properties on an existing text object. Supports text, x, y, fontSize, color, fontFamily.

updateTextObject(myText, { text: 'New text', x: 100, y: 200 });
deleteTextObject (id)

Removes a text object from the overlay and control panel.

deleteTextObject(myText);

Colors

setCharacterColors (id, colors[])

Sets a tint color on each individual character. The array must match the full text length exactly — spaces count as characters.

Base text color should be white (0xFFFFFF) for tint to render correctly. Tint multiplies against the base color.
setCharacterColors(myText, [
  0xff0000, // H
  0xff8800, // e
  0xffff00, // l
  0x00ff00, // l
  0x0088ff, // o
]);
randomColor () → number

Returns a random color as a hex number.

const col = randomColor(); // e.g. 0xa3f2c1

Effects

waveEffect (id, amplitude=30, frequency=0.5, speed=0.1) → intervalId

Animates a sine wave through the characters on the Y axis. Runs continuously.

Param Default Description
amplitude 30 Wave height in pixels. Lower = more subtle.
frequency 0.5 How many waves across the text. Lower = longer wave.
speed 0.1 How fast the wave moves. Lower = slower. 
waveEffect(myText); // default
waveEffect(myText, 8, 0.4, 0.05); // subtle
bounceEffect (id, height=50, speed=0.1) → intervalId

Each character bounces with a stagger delay between them.

Param Default Description
height 50 Bounce height in pixels.
speed 0.1 Bounce speed. 
bounceEffect(myText, 30, 0.08);
shakeEffect (id, intensity=5) → intervalId

Applies random X/Y jitter to every character at 50ms intervals.

Param Default Description
intensity 5 Max pixel displacement in each direction. 
shakeEffect(myText, 3); // subtle tremor
spiralInEffect (id, radius=40, speed=0.06, stagger=0.6, linger=0) → intervalId

Characters spiral in one by one from an orbit radius down to their resting position. Loops continuously.

Param Default Description
radius 40 Orbit radius in pixels.
speed 0.06 Spiral speed.
stagger 0.6 Delay between each character starting.
linger 0 Ms to pause at rest between cycles. 
spiralInEffect(myText, 40, 0.06, 0.6, 1000); // 1s pause
extrudeEffect (id, distance=6, angle=Math.PI/4, color=0x000000, blur=0)

Adds a drop shadow that creates a 3D extrude look.

Param Default Description
distance 6 Shadow offset distance in pixels.
angle Math.PI/4 Shadow angle in radians. PI/4 = bottom-right.
color 0x000000 Shadow color.
blur 0 Shadow blur amount. 
extrudeEffect(myText, 8, Math.PI / 4, 0x222222);
jitterBaseline (id, intensity=2)

One-time random Y nudge per character. Baked in and stays consistent — call once before your animation to give text an organic, handset feel.

Param Default Description
intensity 2 Max Y displacement in pixels. 
jitterBaseline(myText, 3);
waveEffect(myText); // wave plays over the baked jitter
setCharacterSize (id, sizes[])

Sets a scale multiplier on each individual character. 1 = normal size.

setCharacterSize(myText, [1.5, 1, 1, 1, 0.8]);
setCharacterOffsets (id, offsets[])

Manually sets X/Y offset for each character. Used internally by wave, bounce, etc. You can call it directly for custom effects.

setCharacterOffsets(myText, [
  { x: 0, y: -10 },
  { x: 5, y: 0 },
  { x: 0, y: 10 }
]);

Transitions

fadeIn (id, duration=1000) → intervalId

Fades the text in over the given duration in milliseconds.

fadeIn(myText, 500);
fadeOut (id, duration=1000) → intervalId

Fades the text out over the given duration in milliseconds.

fadeOut(myText, 800);

Timers

Use these instead of native setTimeout / setInterval. They are automatically cleaned up when the script stops.
setTimeout (callback, delay) 
setTimeout(() => {
  updateTextObject(myText, { text: 'Changed!' });
}, 2000);
setInterval (callback, delay) → intervalId 
const id = setInterval(() => {
  // runs every 16ms (~60fps)
}, 16);

Drawing

drawTriangle (x1,y1, x2,y2, x3,y3, color)

Draws a single filled triangle in world space.

drawTriangle(100,100, 200,300, 0,300, 0xff0000);
drawTriangles (triangles[])

Batch version of drawTriangle. Preferred for animation frames — more efficient than calling drawTriangle in a loop.

drawTriangles([
  { x1:0, y1:0, x2:100, y2:0, x3:50, y3:100, color:0x00ff00 },
  { x1:200, y1:0, x2:300, y2:0, x3:250, y3:100, color:0x0088ff }
]);
clearTriangles ()

Clears all drawn triangles. Call at the start of each animation frame.

setInterval(() => {
  clearTriangles();
  drawTriangles(myFrame);
}, 16);

Animation Loop

startAnimation (callback)

Runs a callback in sync with the PixiJS ticker (every frame). Use for smooth, frame-perfect animations.

startAnimation(() => {
  // called every frame
});
stopAnimation () 
stopAnimation();

Console

Logs messages to the script console panel.

console.log('hello');
console.warn('watch out');
console.error('something broke');

Examples

Lissajous Curve setCharacterOffsets + setInterval

Each character follows a Lissajous curve — a parametric path defined by two sine waves at different frequencies. The result is a looping figure-8 style motion across the whole word.

const myText = createTextObject({
  text: 'MATHEMATICALVISUALIZATION',
  x: 640, y: 360,
  fontSize: 46,
  color: 0xFFFF00
});

let time = 0;
setInterval(() => {
  const offsets = [];
  const textLength = 25;

  for (let i = 0; i < textLength; i++) {
    const t = (i / textLength) * Math.PI * 2 + time;

    // Lissajous: x = A*sin(at + δ), y = B*sin(bt)
    const x = 80 * Math.sin(3 * t + Math.PI / 4);
    const y = 60 * Math.sin(2 * t);

    offsets.push({ x, y });
  }

  setCharacterOffsets(myText, offsets);
  time += 0.05;
}, 16);
Double Helix setCharacterOffsets + setInterval

Characters follow a double helix path using sine and cosine. The Z axis is simulated as an X offset, giving a sense of depth as characters rotate through 3D space.

const myText = createTextObject({
  text: 'PINKELEPHANTSONPARADE',
  x: 640, y: 360,
  fontSize: 52,
  color: 0xFF00FF
});

let time = 0;
setInterval(() => {
  const offsets = [];
  const textLength = 21;

  for (let i = 0; i < textLength; i++) {
    const t = (i / textLength) * Math.PI * 6 + time;

    // Double helix — Z simulated as X offset for depth
    const x = Math.cos(t) * 30;
    const y = Math.sin(t) * 60;

    offsets.push({ x, y });
  }

  setCharacterOffsets(myText, offsets);
  time += 0.08;
}, 16);

3D Examples

Rotating Cube rotateY + rotateX + project3D + drawTriangles

A cube defined by 8 vertices, rotated on two axes every frame and drawn as 12 triangles (2 per face). Each face has its own color.

const vertices = [
  [-100, -100, -100], [100, -100, -100], [100, 100, -100], [-100, 100, -100],
  [-100, -100,  100], [100, -100,  100], [100,  100,  100], [-100,  100,  100]
];

let angle = 0;

startAnimation(function() {
  const projected = vertices.map(v => {
    let r = rotateY(v[0], v[1], v[2], angle);
    r = rotateX(r.x, r.y, r.z, angle * 0.5);
    return project3D(r.x, r.y, r.z);
  });

  drawTriangles([
    // Front - red
    { x1:projected[4].x, y1:projected[4].y, x2:projected[5].x, y2:projected[5].y, x3:projected[6].x, y3:projected[6].y, color:0xFF0000 },
    { x1:projected[4].x, y1:projected[4].y, x2:projected[6].x, y2:projected[6].y, x3:projected[7].x, y3:projected[7].y, color:0xFF0000 },
    // Back - dark red
    { x1:projected[0].x, y1:projected[0].y, x2:projected[1].x, y2:projected[1].y, x3:projected[2].x, y3:projected[2].y, color:0x880000 },
    { x1:projected[0].x, y1:projected[0].y, x2:projected[2].x, y2:projected[2].y, x3:projected[3].x, y3:projected[3].y, color:0x880000 },
    // Right - orange
    { x1:projected[1].x, y1:projected[1].y, x2:projected[5].x, y2:projected[5].y, x3:projected[6].x, y3:projected[6].y, color:0xFF8800 },
    { x1:projected[1].x, y1:projected[1].y, x2:projected[6].x, y2:projected[6].y, x3:projected[2].x, y3:projected[2].y, color:0xFF8800 },
    // Left - brown
    { x1:projected[0].x, y1:projected[0].y, x2:projected[4].x, y2:projected[4].y, x3:projected[7].x, y3:projected[7].y, color:0xCC6600 },
    { x1:projected[0].x, y1:projected[0].y, x2:projected[7].x, y2:projected[7].y, x3:projected[3].x, y3:projected[3].y, color:0xCC6600 },
    // Top - yellow
    { x1:projected[3].x, y1:projected[3].y, x2:projected[2].x, y2:projected[2].y, x3:projected[6].x, y3:projected[6].y, color:0xFFFF00 },
    { x1:projected[3].x, y1:projected[3].y, x2:projected[6].x, y2:projected[6].y, x3:projected[7].x, y3:projected[7].y, color:0xFFFF00 },
    // Bottom - olive

  ]);

  angle += 0.02;
});
Rotating Pyramid rotateY + rotateX + project3D + drawTriangles

A pyramid with a square base and single apex. Position-shifted to upper-right via manual X/Y offset on the projected coordinates. Each face a different color.

const pyramidVertices = [
  [-80,  80, -80],  // 0: bottom-left
  [ 80,  80, -80],  // 1: bottom-right
  [ 80,  80,  80],  // 2: back-right
  [-80,  80,  80],  // 3: back-left
  [  0, -100,   0]   // 4: apex
];

let pyramidAngle = 0;

startAnimation(function() {
  const projected = pyramidVertices.map(v => {
    let r = rotateY(v[0], v[1], v[2], pyramidAngle);
    r = rotateX(r.x, r.y, r.z, pyramidAngle * 0.3);
    const p = project3D(r.x, r.y, r.z);
    // Shift to upper-right
    return { x: p.x + 400, y: p.y - 200 };
  });

  drawTriangles([
    // Base
    { x1:projected[0].x, y1:projected[0].y, x2:projected[1].x, y2:projected[1].y, x3:projected[2].x, y3:projected[2].y, color:0x4488FF },
    { x1:projected[0].x, y1:projected[0].y, x2:projected[2].x, y2:projected[2].y, x3:projected[3].x, y3:projected[3].y, color:0x4488FF },
    // Front
    { x1:projected[0].x, y1:projected[0].y, x2:projected[1].x, y2:projected[1].y, x3:projected[4].x, y3:projected[4].y, color:0x00FFFF },
    // Right
    { x1:projected[1].x, y1:projected[1].y, x2:projected[2].x, y2:projected[2].y, x3:projected[4].x, y3:projected[4].y, color:0x00FF88 },
    // Back
    { x1:projected[2].x, y1:projected[2].y, x2:projected[3].x, y2:projected[3].y, x3:projected[4].x, y3:projected[4].y, color:0x88FF00 },
    // Left
    { x1:projected[3].x, y1:projected[3].y, x2:projected[0].x, y2:projected[0].y, x3:projected[4].x, y3:projected[4].y, color:0xFFFF00 },
  ]);

  pyramidAngle += 0.03;
});
